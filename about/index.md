# About

I'm Deepanshu Kartikey, a Performance Engineer at
[Clickpost](https://clickpost.ai) working on the platform team.
I go deep into systems to make them faster.

Outside of work I contribute to the Linux kernel — fixing bugs in filesystems
(ext4, ntfs3, ocfs2, f2fs, gfs2, btrfs, romfs), memory management
(hugetlbfs, mm/thp, mm/slub, mm/vmalloc, mm/memfd), networking
(atm/lec, net/nfc, net/usb, wifi/mac80211), and core subsystems
(kernel/fork, bpf, block, iommufd, comedi, tracing).
Most of my bugs come from [syzbot](https://syzkaller.appspot.com).

All my merged patches are on the
[kernel.org git log](https://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux.git/log/?qt=author&q=deepanshu).
